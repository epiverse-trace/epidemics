# Validate objects that inherit from the class

Validate objects that inherit from the class

Validate objects that inherit from the class

## Usage

``` r
validate_contacts_intervention(x)

validate_rate_intervention(x)
```

## Arguments

- x:

  For `validate_contacts_intervention()`, an object to be validated as a
  `<contacts_intervention>`. For `validate_rate_intervention()`, an
  object to be validated as a `<rate_intervention>`.

## Value

Invisibly returns the input `x`. Called primarily for its input checking
as a validator for objects of the `<intervention>` superclass.
